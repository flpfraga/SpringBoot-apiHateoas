Projeto de desenvolvimento de uma API ReSTFull com Java e Spring Boot

A API fornece endpoints para Pessoas, podendo ser resgatados registros salvos, adicionar novos, atualização de existentes e exclusão.

A API tem conexão com banco de dados MySql. Utiliza do JPA e Hibernate do SpringData.

Foi implementado funcionalidade de Migrations através do FlyWay.

API com suporte de content negotiation, podendo fornecer e receber os dados em formato json, xml e x-yaml.

Implementado funcionalidade Hateoas.

Para todas funcionalidades acimas foram implementados testes unitários.

API com suporte ao swagger.
